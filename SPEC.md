# 身心健康虚拟宠物 - 项目规格说明

## 1. Concept & Vision

打造一个**有生命的2D虚拟宠物**，它不仅是一个静态图标，而是一个能呼吸、会思考、有情感的数字化生命体。宠物通过细腻的视觉表现（呼吸起伏、眼神变化，光影流动）和丰富的行为模式（自由走动、随机动作、情感响应）成为用户真正的数字陪伴伙伴。

设计理念：**"数字生命，轻盈陪伴"** —— 宠物不是工具，而是一个有温度的存在。

### 字体
- **主字体**：系统emoji字体
- **中文字体**：Noto Sans SC
- **备用**：-apple-system, "Segoe UI", sans-serif

### 空间系统
- 浮动宠物窗口：270x300px（1.5倍放大）
- 周围呼吸空间：至少20%
- 粒子分布半径：宠物大小的1.5倍
- 轨迹点最大数量：200个

### 动效哲学
- **呼吸**：3-5%身体膨胀，4秒周期
- **眨眼**：2-6秒随机间隔，0.15秒完成
- **移动**：ease-out缓动，8-12秒穿越半张地图
- **表情过渡**：300ms cubic-bezier(0.4, 0, 0.2, 1)
- **轨迹渐隐**：30秒生命周期

## 3. 宠物形象设计

### 基础结构（4种宠物）：

#### 🐰 Bunny（兔子）
- **体型**：高瘦圆润，耳朵占身高40%
- **特征**：长耳朵带粉色内耳、圆尾巴
- **颜色**：主色 + 粉色内耳 + 白肚子

#### 🐱 Cat（猫咪）
- **体型**：中等身材，尾巴飘逸
- **特征**：尖耳朵、胡须、条纹
- **颜色**：主色 + 深色条纹 + 白色脸部

#### 🐻 Bear（小熊）
- **体型**：矮胖圆润
- **特征**：圆耳朵、圆鼻子、短四肢
- **颜色**：主色 + 浅色脸部/肚子

#### 🦊 Fox（狐狸）
- **体型**：修长优雅
- **特征**：尖耳朵、大尾巴、尖脸
- **颜色**：主色 + 白色脸部/尾巴尖

## 5. 动画系统

### 待机动画（必须）
1. **呼吸起伏**：身体Y轴缩放3-5%，4秒周期
2. **随机眨眼**：2-6秒间隔，0.15秒眨眼
3. **轻微摇晃**：整体旋转±2°，6秒周期
4. **耳朵/尾巴**：跟随呼吸微动
5. **光效呼吸**：边缘光晕明暗变化

### 交互动画
1. **点击反应**
   - 弹跳：scale 1→1.15→0.95→1，0.4秒
   - 表情切换开心：200ms过渡
   - 粒子爆发：8-12个向外扩散
   - 气泡显示：2秒后消失

2. **连续点击彩蛋**
   - 第1-3次：正常弹跳+随机问候
   - 第4次：原地转圈1圈
   - 第5次：触发彩虹星星雨

3. **自由走动AI**
   - 随机目标点（页面内）
   - 移动速度：8-12秒穿越半张地图
   - 到达后随机停留8-20秒
   - CSS transition平滑移动

4. **画画轨迹**
   - 宠物移动时在身后留下情绪轨迹
   - 不同心情产生不同颜色和形状
   - 轨迹随时间逐渐变淡消失（约30秒）

## 6. 技术实现

### 架构
- **宠物渲染**：SVG主体 + CSS动画
- **粒子系统**：Canvas叠加层
- **轨迹系统**：独立Canvas图层绘制情绪轨迹
- **AI移动**：CSS transition + 定时器
- **状态管理**：集中式State对象

### 性能优化
- 粒子上限：15个
- 轨迹点最大数量：200个
- will-change属性优化
- 懒渲染（页面隐藏时暂停动画）
- 页面可见性控制

### 数据持久化
- localStorage保存：
  - 宠物类型、颜色、名字
  - 心情值、压力值、能量值
  - 愉快值（隐藏）
  - 宠物位置(x, y)
  - 倾诉记录（最近200条）
  - 首次访问标记

## 8. 2.5D重庆卡通地图设计

### 标志性元素
1. **洪崖洞** - 层叠吊脚楼，11层轮廓，暖黄色灯光闪烁
2. **来福士广场** - 6座弧形塔楼，银白色玻璃反光
3. **长江/嘉陵江** - 蓝色曲线河流，微光流动动画
4. **跨江大桥** - 千厮门大桥、渝澳大桥，桥索细节
5. **轨道交通** - 卡通轻轨车厢穿过
6. **大剧院** - 紫色穹顶建筑

### 情绪区域
| 区域 | 地标 | 氛围 | 宠物行为 |
|------|------|------|---------|
| 温暖区 | 洪崖洞、来福士 | 繁华热闹 | 偶尔停留，表情变开心 |
| 活力区 | 轻轨、商场 | 动感活力 | 快速穿过 |
| 宁静区 | 江边步道 | 平静祥和 | 沿河漫步，表情变平静 |
| 放松区 | 大树、草坪 | 宁静治愈 | 长时间停留 |

### 地图动画
- 云朵缓慢飘动（CSS animation）
- 河水流动（SVG animate）
- 窗户灯光闪烁（SVG animate）
- 轻轨车厢移动（SVG animateTransform）
- 花朵摇摆（CSS animation）
- 树木微动（CSS animation）
